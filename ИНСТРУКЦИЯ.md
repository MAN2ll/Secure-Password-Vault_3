# Secure Vault — Установка без компьютера

## Шаг 1 — GitHub (бесплатно)
1. Откройте https://github.com/signup с телефона
2. Нажмите «+» → «New repository»
3. Имя: SecureVault, тип: **Public**, нажмите «Create»

## Шаг 2 — Загрузите файлы
Нажмите «uploading an existing file», загрузите ВСЕ файлы из архива, сохраняя структуру папок.

## Шаг 3 — Получите APK
1. Вкладка «Actions» → дождитесь зелёной галочки (5-10 мин)
2. Справа нажмите «Releases» → «latest»
3. Нажмите **SecureVault.apk** — скачается напрямую

## Шаг 4 — Установка
Настройки → Безопасность → Установка из неизвестных источников → Разрешить
Откройте APK → Установить

## Требования: Android 8.0+
